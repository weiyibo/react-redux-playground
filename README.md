# react-starter-kit

It contains the following libs:

* React
* React Redux
* React Router,
* React Redux Router, 
* Bootstrap
* Bootswatch (Cosmo be default, because I like it)
* Less
* Busy Indicator
* Webpack
* ExtractPlugin (Extract css into separate file)
* react-bootstrap
* merge
* moment-tz

# Run

```
npm install
npm start
```

# Deplay

```
npm run package
```

The files will be in **Build** folder
